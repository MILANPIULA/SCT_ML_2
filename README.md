# Customer Segmentation using K-Means Clustering

## Project Overview
This project uses the K-Means Clustering algorithm to segment mall customers based on their Annual Income and Spending Score.

## Dataset
- Mall_Customers.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm
- K-Means Clustering

## Output
The model groups customers into 5 clusters and visualizes the customer segments with cluster centroids.
